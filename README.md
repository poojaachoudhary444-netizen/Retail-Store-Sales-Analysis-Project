# Retail Store Sales Analysis Project

## Project Overview

This project analyzes retail store sales data using SQL and Power BI to generate business insights related to sales performance, customers, products, inventory, and store operations.

The project includes database creation, SQL analysis, KPI generation, and interactive Power BI dashboards.


# Tools & Technologies Used

* SQL Server Management Studio (SSMS)
* SQL
* Power BI
* Excel

* Analyze sales performance

# Key Insights

* Baldwin Bikes generated the highest sales.
* Trek products contributed the highest revenue.
* Sales peaked during 2017.
* Some products showed low stock levels despite high demand.
* Customer purchasing behavior varied across stores and brands.
